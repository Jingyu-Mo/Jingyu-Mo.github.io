Markdown Tutorial

发表于 2016-12-26 | 分类于 编程 | | 阅读次数

Markdown 是一种兼顾可读性与易用性的轻量级标记语言,由约翰·格鲁伯（John Gruber）和亚伦·斯沃茨（Aaron Swartz）创建。它可以将文本转化为HTML。对于一些不常用的标记，Markdown 允许直接将HTML标记插入文本。
